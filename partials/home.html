Travis Wall

I'm a SaaS which gives you feedbacks about your Travis!

Hey everyone! I'm Travis Wall, an Open Source project firstly built to discover AngularJS to Eric GELOEN (my creator) but at the end, I was so pretty and amazing than he has decided to make me discover the world!

What's inside?

I was built around many technologies but the most important ones are: AngularJS, Masonry, MomentJS, Less, LessHat, Normalize CSS, Bower, Grunt, Protractor, Jasmine, Karma (just for the awesome), Travis-CI (obviously) and many others...

How does I work?

Every 15 seconds, I made an HTTP request to the Travis API and update myself accordingly. Each calls to the Travis API needs to be authenticated using a public token (for public repositories) and/or a private token (for private repositories).

To generate your tokens, install the Travis CLI tool and use the following commands:

  • Public Token: travis login && travis token.
  • Private Token: travis login --pro && travis token --pro.
